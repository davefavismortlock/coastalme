Detailed Description

Geometry class used to represent 2D point objects with floating-point co-ordinates.

The CGeom2DPoint class is used to represent 2D points where the x and y co-ordinates are floating-point values, e.g. points for which the x and y co-ordinates are in the external CRS (co-ordinate reference system)
